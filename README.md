# Customer Retention Cohort Analysis

This project demonstrates how Product Managers, Growth Teams, and Data Analysts can use customer event data to understand user retention, churn, activation, and long-term engagement.

Using a realistic SaaS dataset, the analysis answers critical business questions:

- Are users returning after signup?
- Which cohorts retain best over time?
- Which acquisition channels bring high-quality customers?
- What user behaviors predict long-term retention?
- What is the revenue impact of improving retention?

## Key Analyses

- Data Cleaning & Validation
- Cohort Analysis
- Retention Matrix Generation
- Retention Heatmaps
- Retention Curve Visualization
- Channel Quality Analysis
- User Journey Funnel Analysis
- Power User Segmentation
- Time-to-First-Value (TTFV) Analysis
- Invite vs Engagement Analysis
- Revenue Impact Modeling

## Business Outcome

The project converts raw user activity logs into actionable product and growth recommendations, helping teams identify churn risks, improve activation, and prioritize high-impact retention initiatives.
